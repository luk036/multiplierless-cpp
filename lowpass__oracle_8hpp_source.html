12// Modified from CVX code by Almir Mutapcic in 2006.
13// Adapted in 2010 for impulse response peak-minimization by convex iteration by
14// Christine Law.

16// "FIR Filter Design via Spectral Factorization and Convex Optimization"
17// by S.-P. Wu, S. Boyd, and L. Vandenberghe
18//
19// Designs an FIR lowpass filter using spectral factorization method with
20// constraint on maximum passband ripple and stopband attenuation:
21//
22// minimize max |H(w)| for w in stopband
23// s.t. 1/delta <= |H(w)| <= delta for w in passband
24//
25// We change variables via spectral factorization method and get:
26//
27// minimize max R(w) for w in stopband
28// s.t. (1/delta)**2 <= R(w) <= delta**2 for w in passband
29// R(w) >= 0 for all w
30//
31// where R(w) is squared magnitude frequency response
32// (and Fourier transform of autocorrelation coefficients r).
33// Variables are coeffients r and G = hh' where h is impulse response.
34// delta is allowed passband ripple.
35// This is a convex problem (can be formulated as an SDP after sampling).
36
