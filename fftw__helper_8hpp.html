Detailed Description

FFTW wrappers and complex array ops for spectral factorization.

Provides FFT/IFFT wrappers around the FFTW3 library plus complex array arithmetic operators. These are kept separate from the core Arr type (which now lives in ellalgo-cpp/include/ellalgo/arr.hpp).
